# AgencyOS

A full-featured agency management OS template for Directus. Includes CRM, project management, invoicing, proposals, help desk, and a marketing website with a block builder.

## Getting Started

### Using Directus Cloud

1. Create a new project at [Directus Cloud](https://directus.io/cloud/) and select the **AgencyOS** template.
2. Generate a static token for the admin user.

### Using Docker (Local)

```bash
npx directus-template-cli@latest init
```

Select **AgencyOS** from the template list and follow the prompts.

## What's Included

- Contacts, organizations, and deals
- Projects, tasks, and time tracking
- Invoices and expenses
- Proposals with approval workflows
- Help desk with articles and feedback
- Marketing website with page builder
- Dashboards and analytics
